üîå API Endpoints

Use these endpoints to integrate payments into your applications.

GET /api/health

Health check endpoint

curl https://your-app.vercel.app/api/health

GET /api/widget

Widget HTML page

curl https://your-app.vercel.app/api/widget

POST /api/donate

Process a donation payment

curl -X POST https://your-app.vercel.app/api/donate \ -H "Content-Type: application/json" \ -d '{"amount": 5}'

POST /api/pay-service

Process AI agent service payment

curl -X POST https://your-app.vercel.app/api/pay-service \ -H "Content-Type: application/json" \ -d '{"serviceId": "ai-service", "amount": 10, "walletAddress": "0x..."}'
